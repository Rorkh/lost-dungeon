### Hosting:
If you want to host LD:
1. git clone https://github.com/vladikvt/lost-dungeon
2. Rename 'databases/accountsEXAMPLE' to 'databases/accounts'
3. Run 'python3 main.py'
4. Run 'telnet [server ip] 4000'
